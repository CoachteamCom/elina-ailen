# Elina Ailen Portfolio

Fashion designer portfolio website.

## Deployment via GitHub Desktop

### Stap 1: Nieuwe repository aanmaken
1. Open GitHub Desktop
2. Klik **File → New Repository**
3. Name: `elina-portfolio`
4. Local Path: `C:\Users\[jouw-naam]\Documents\GitHub`
5. Klik **Create Repository**

### Stap 2: Files toevoegen
1. Kopieer `index.html` en `vercel.json` naar de nieuwe map
2. In GitHub Desktop zie je nu "2 changed files"
3. Onderaan typ je een commit message: `Initial portfolio`
4. Klik **Commit to main**
5. Klik **Publish repository** (rechtsboven)
6. Vink "Keep this code private" UIT als je Vercel free wilt gebruiken
7. Klik **Publish Repository**

### Stap 3: Vercel koppelen
1. Ga naar [vercel.com](https://vercel.com)
2. Log in met GitHub
3. Klik **Add New → Project**
4. Selecteer **elina-portfolio** repository
5. Klik **Deploy**
6. Klaar! Je krijgt een URL zoals `elina-portfolio.vercel.app`

### Custom domain (optioneel)
In Vercel dashboard → Settings → Domains → voeg toe: `elinaailen.com`

## Quick Edit Guide

De hele website zit in `index.html`. Om tekst aan te passen:

- **Naam**: Zoek naar `Elina` en `Ailen` in de hero sectie
- **Tagline**: Zoek naar `"Where Victorian elegance...`
- **Bio teksten**: Zoek naar `About the Designer`
- **Collectie naam**: Zoek naar `Susurros del Pasado`
- **Contact**: Zoek naar `elina@elinaailen.com` en `@elicardozo1`

Na wijzigingen:
1. Save het bestand
2. In GitHub Desktop: commit + push
3. Vercel update automatisch binnen 1 minuut
